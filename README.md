# 前言
&nbsp;&nbsp;&nbsp;&nbsp;最近想做一个业务简单的系统,夯实下基础(前端+后端)。人力资源系统业务上较为简单,也比较独立,适合个人练手项目。人力资源系统包括一些核心的人力资源管理业务功能如:**考勤管理**、**薪酬管理**、**绩效管理**、**员工管理**、**加班申请**、**出差申请**等模块。

# 开发环境
